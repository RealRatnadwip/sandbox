Python Functions Notes

Simple notes for understanding Python functions.

1. What is a Function?

A function is a block of reusable code used to perform a specific task. Functions help reduce repetition and make programs easier to manage. 

def greet():
    print("Hello")

2. Calling a Function

After defining a function, we call it using its name followed by brackets.

greet()

3. Function with Parameters

Parameters allow us to pass values into a function. 

def add(a, b):
    print(a + b)

add(5, 3)

4. Return Statement

The return keyword sends a value back from the function. 

def square(n):
    return n * n

result = square(4)
print(result)

5. Types of Functions

  • Built-in Functions → print(), len(), input()
  • User-defined Functions → Functions created by the programmer

6. Advantages of Functions

  • Code Reusability
  • Easy Debugging
  • Better Readability
  • Reduces Code Length

7. Example Program

def factorial(n):
    fact = 1

    for i in range(1, n + 1):
        fact = fact * i

    return fact

num = 5
print("Factorial =", factorial(num))
