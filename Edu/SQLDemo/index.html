SQL Commands Reference with Examples

DDL Commands

  • CREATE — Create table/database
  • ALTER — Modify structure
  • DROP — Delete table
  • TRUNCATE — Remove all data
CREATE TABLE Students ( id INT PRIMARY KEY, name VARCHAR(50), age INT );

Example Table

IDNameAge
1John20
2Alice22

DML Commands

INSERT INTO Students (id, name, age) VALUES (1, 'John', 20);UPDATE Students SET age = 21 WHERE id = 1; DELETE FROM Students WHERE id = 1;

DQL (SELECT)

SELECT * FROM Students;SELECT name, age FROM Students WHERE age > 18;

Joins

SELECT s.name, c.course_name FROM Students s INNER JOIN Courses c ON s.id = c.course_id;

Advanced Topics

  • INDEX — Improves query speed
  • VIEW — Virtual table
  • TRIGGER — Auto-execute on events
  • PROCEDURE — Stored SQL logic

Index Example

CREATE INDEX idx_name ON Students(name);

View Example

CREATE VIEW adult_students AS SELECT * FROM Students WHERE age > 18;

Trigger Example

CREATE TRIGGER before_insert BEFORE INSERT ON Students FOR EACH ROW SET NEW.name = UPPER(NEW.name);

Stored Procedure

CREATE PROCEDURE GetStudents() BEGIN SELECT * FROM Students; END;
