Polylines & Arcs

An Interactive Guide for AutoCAD 2015

1

The Power of the Polyline (PLINE)

A Polyline is a sequence of connected line or arc segments treated as a single object. This is ideal for creating complex outlines that you can easily select, move, or offset as one entity. Think of it as drawing a complete shape without lifting your pen.

How to Use the PLINE Command

> Command: PLINE (or PL)

1. Start the Command: Type PLINE and press Enter.

2. Specify Start Point: Click anywhere in your drawing area to set the first point.

3. Specify Next Point: Click to define the end of the first segment. AutoCAD will prompt for the next point, allowing you to continue drawing.

4. Close or End: Press Enter to end, or type C and Enter to draw a final segment back to the start point, creating a closed object.

2

Integrating Arcs into a Polyline

This is the most efficient way to create objects with both straight and curved edges. While inside the `PLINE` command, you can seamlessly switch between drawing lines and arcs.

Example: Drawing a Keyhole Shape

1 Start PLINE
2 Draw LINE segment
3 Switch to ARC mode ('A')
4 Draw ARC segment
5 Switch to LINE mode ('L')
6 CLOSE shape ('C')

This diagram shows the command flow for creating the shape.

1. Start & Draw Base: Begin `PLINE`, click a start point, move right, type `50` and press Enter.

2. Draw Right Side: Move cursor up, type `50` and press Enter.

Specify next point or [Arc/...]: A

3. Switch to Arc Mode: Type `A` and press Enter.

4. Create the Top Arc: Move your cursor 50 units to the left (above the start point) and click. This completes the semi-circle.

Specify endpoint of arc or [.../Line/...]: L

5. Switch back to Line Mode: Type `L` and press Enter.

Specify next point or [.../Close/...]: C

6. Close the Shape: Type `C` and press Enter to draw the final line segment back to the start.

3

The Standalone ARC Command

For times you need a single arc not part of a polyline. AutoCAD offers many ways to define an arc based on the information you have (e.g., start/end points, center, radius, angle).

Accessing the ARC tool

> Command: ARC (or A)

Find it on the Ribbon: Home tab > Draw panel > Arc.

Common Creation Methods

  • 3-Point (Default): Click three points.
  • Start, Center, End: Define by start, center, and end points.
  • Start, End, Radius: Define by start, end, and a radius.
  • Start, End, Angle: Define using start, end, and included angle.
4

Pro-Tip: Editing & Joining (PEDIT)

Combine separate, end-to-end lines and arcs into a single polyline with the powerful `PEDIT` (Polyline Edit) command. This is essential for cleaning up drawings.

How to Join Objects into a Polyline

> Command: PEDIT

1. Start Command: Type `PEDIT` and press Enter.

2. Select Object: Click one line/arc. If it's not a polyline, AutoCAD will ask to convert it. Press Enter to accept.

3. Choose Join Option: Type `J` for Join and press Enter.

4. Select Objects to Join: Select all other lines and arcs to combine.

5. Finish: Press Enter twice to complete. Your separate objects are now a single, unified polyline!
