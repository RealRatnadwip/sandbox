# Sandbox

A central repository for various small projects, modules, educational content, and experiments.

## 🗂️ Online File Explorer

This repository features a dynamic, web-based file explorer built entirely with HTML, CSS, and vanilla JavaScript. It uses the GitHub API to automatically fetch and display the live contents of this repository.

You can browse the available folders and files directly from your browser by visiting the **[Sandbox Directory](https://realratnadwip.github.io/sandbox/)**.
